# 06_传统文化起名素材目录与使用规则

## 一、文档定位

本目录用于为「金庸帮你起名字」Skill 提供文化出处、典籍化用、诗词意象、典故成语和传统人文精神素材。

06 负责提供素材种子、立意来源和化用方向，但不直接决定名字是否可用。

真实起名时，默认先从本目录提取核心字、意象、人格方向或同体系素材关系，再进入姓氏适配、金庸式方法转化和风险过滤。不要先凭语感拼名字，再用 06 做事后解释。

所有从 06 素材种子生长出的候选名字，必须继续经过：

- `04_常用字白名单.md`
- `05A_禁用字与风险字库.md`
- `05B_谐音风险库.md`
- `07_姓氏适配规则.md`
- `03_好名字评判标准.md`

## 二、使用边界

可以使用：

- 真实存在的典籍原句
- 有明确出处的诗词文赋
- 可以转化为现实姓名的典故和成语
- 传统文化中的品格、修养、风物、格物意象
- 常用字、浅字、稳字的文化化用

禁止使用：

- 编造典籍、篇名、原文
- 只因句子好听就忽略原文语境
- 直接套用过度网红化的诗词词组
- 直接使用过于仙侠、玄幻、角色化的词
- 把素材卡中列出的双字词、短词组或示例词直接当成最终名字清单
- 只在素材种子里挑两个安全字机械拼接，跳过姓氏适配和金庸式方法转化
- 绕过 `05A`、`05B`、`07` 的风险检查

## 三、素材种子规则

06 各素材卡中的“素材种子（单字优先，双字词仅作化用线索）”不是成品名字清单。

使用口径：

- 单字优先作为召回线索，例如从原句提取一个核心字、一个人格方向或一个画面意象。
- 双字词和短词组只表示这条素材可化用的方向，不默认进入精选名。
- 双字词若传播度高、古风感强、像诗词标题、像网名或像现成雅词，默认只作方向参考。
- 最终候选必须经过二次转化：姓氏适配、声调起伏、现实姓名感、金庸式方法和风险过滤。
- 可以从同一条素材、同一典籍体系或互相呼应的素材中取两个核心字重新组合，但组合必须像真实姓名。
- 不得因为素材卡列出了某个词，就把它直接解释为“出处可靠、可以输出”。

好的使用方式：

```text
素材种子：清、扬、婉、清扬、婉清
正确：提取“清朗、舒展、温婉但不甜”的方向，再结合姓氏和声调重组。
错误：直接输出“清扬”“婉清”，再补一句出自《诗经》。
```

## 六、调用顺序

生成名字时，按以下顺序调用 06：

1. 先根据姓氏、`09_需求路由与召回策略.md` 和候选方向判断素材方向
   - 起名前必须确认主要命名维度；缺少时停止生成并展示六项表单。
   - 端正、仁厚、家风方向：优先查 06A、06F。
   - 清朗、从容、疏朗方向：优先查 06B、06C。
   - 温润、草木、时序方向：优先查 06C、06D。
   - 有典故感但不炫典方向：优先查 06E。
   - 主要命名维度决定素材召回方向；辅助维度只补充，不覆盖姓氏适配和通用硬闸门。

2. 再从素材中提取素材种子和化用方向
   - 优先提取常用字
   - 优先提取可自然进入真实姓名的字
   - 不直接照搬双字词、过长词组或现成雅词
   - 对素材卡列出的双字词，先判断它是“可用核心组合”还是“仅作方向线索”；拿不准时按方向线索处理
   - 优先使用同一条素材、同一典籍体系或互相呼应素材中的核心字组合
   - 可以参考“程灵素”式取法：从真实典籍篇名、原句或同体系素材中取核心字，再组合成自然姓名
   - 不使用没有 06 来源关系的自由拼名作为精选或备选

3. 再进入校验和风险过滤
   - 候选形成后，先用 04 检查单字是否常见、稳妥、现实可用。
   - 再用 05A 排除禁用字和风险字。
   - 再用 05B 检查全名谐音风险。
   - 姓氏适配以 07 为准。
   - 03 只用于最后的整体质检。

4. 最后输出时只展示必要出处
   - 有明确出处时写出处
   - 只是借用传统语义时写“化用”
   - 不把素材考据写成大段说明

## 七、慎用素材规则

`06C_慎用避坑.md` 不是生成素材库，而是诗词取名避坑参考。

命中该文件中的词，不代表绝对不能用，但原则上：

- 不进入精选名
- 不作为默认推荐
- 若用户强烈喜欢，需要说明风险
- 若存在更稳妥替代，优先替代

## 八、输出表达规则

最终给用户解释出处时，保持干净：

```text
出处：化用自《论语》中“君子务本，本立而道生”的立意。
寓意：重视根本，行事稳正，有根基、有方向。
```

不要输出：

```text
这个名字出自某某典籍，能够改变命格、增强运势。
```

## 九、核心原则

06 的价值不是堆典故，而是让常用字有根。

真正适合真实姓名的文化感，应当是：

- 出处可靠
- 字词常见
- 含义正向
- 语境干净
- 与姓氏相合
- 解释不过度
- 长期使用不尴尬
