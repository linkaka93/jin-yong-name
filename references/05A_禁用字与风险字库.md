# 05A_禁用字与风险字库

## 一、文档定位

本文件是「金庸帮你起名字」Skill 的用字风险控制库。

它用于识别和排除不适合真实姓名的字、字组和风格倾向，优先级高于 `04_常用字白名单.md`。

核心规则：

> 如果某个字同时出现在 `04_常用字白名单.md` 和本文件中，以本文件为准。

本文件处理：

- 绝对禁用字
- 慎用提示字
- 原著强绑定字组
- `04` 扩展草案复核
- 网红化、仙侠化、生僻化、负面化、江湖化等用字风险

本文件不处理：

- 谐音风险：交给 `05B_谐音风险库.md`
- 姓氏适配：交给 `07_姓氏适配规则.md`
- 典籍出处：交给 `06_传统文化起名素材/`
- 最终质检：交给 `03_好名字评判标准.md`

---

## 二、风险等级

### 1. 绝对禁用

默认不进入候选，不进入精选，不进入备选。

即使用户指定，也必须提示风险，不主动推荐。

### 2. 高风险

原则上不进入精选名。

只有用户明确指定、且组合确实可解释时，才可作为讨论对象。

### 3. 慎用

不是原则性错误，但不作为优先候选。

必要时可进入备选名，但必须在现实评估中说明风险。

### 4. 观察

暂不定性。

需要结合姓氏、组合、性别倾向、时代感和最终质检判断。

## 三、绝对禁用字

以下 7 类字默认禁用。

### 1. 网红同质化伪风雅字

定位：

近 10 年新生儿起名重名率高，常用于强行堆砌"风雅感"，同质化严重，容易形成短视频式、模板化、伪文化起名风格。

字表：

```text
梓、涵、轩、墨、辰、宸、诺、汐、兮、玥、芮、晞、婼、妤、梵、皓、宥、佑、孜、姿、渲、萱、欣、浩、翰、霖
```

说明：

- 以上为辨识度低、同质化最严重的网红爆款字，绝对禁用
- `泽`、`怡` 字义稳定，保留为慎用，不进入精选名，见第四节“甜美流行感字”

处理方式：

- 绝对禁用
- 不进入 `04` 精选表
- 不作为普通候选名生成
- 用户强制指定时，提示同质化和流行化风险

### 2. 仙侠网文违和字

定位：

自带玄幻修仙、二次元、游戏 ID、小说主角感，脱离真实生活语境，与金庸式"写实落地"的起名方向相悖。

字表：

```text
烬、玦、璟、珩、琊、弑、魅、煞、罡、劫、玄、尘、渊、夙、魑、魍、魉、仙、魔、妖、神
```

处理方式：

- 绝对禁用
- 真实姓名场景不使用
- 不用"有古风感""有仙气"作为保留理由

### 3. 生僻难认难写字

定位：

多数人无法一眼认读或顺利书写，容易造成社交尴尬、系统录入困难、办事不便。真正的文化感不靠生僻字堆砌，金庸式起名强调"浅字深用"。

#### 三叠 / 叠字生僻类

```text
燚、垚、淼、犇、骉、羴、麤、鱻、猋、叒、壵、尛、惢、掱、瞐、畾、厽、芔
```

#### 生僻玉旁类

多被赋予"美玉"寓意，是起名生僻字高发区，日常认读率低，容易读错、写错。

```text
玚、珵、瑒、玦、珩、璟、琀、瑀、玓、玏、珝、瑱、璆、玭、珹、琤、璠、玙、璘、瑢、琋、珒、玿、瑎、琂、玼、璅、瓘、玽、琡、瑄
```

#### 生僻日 / 光字旁类

多寓意"光明、兴盛"，字形看似简单，但读音陌生，容易认错、读错。

```text
旻、昱、昪、昫、晞、曌、昶、昉、暘、曛、晅、昺、曜、昜、曀、曒、曈、曚、昍、烜
```

#### 生僻草木风物类

多取自草木意象，看似清雅，实际辨识度低。

```text
芃、莯、莜、荼、蓼、蓁、蕲、蒴、芪、苕、苼、苾、蔹、蘩、芓、芫、蘅、菀、芣、苡、莛、荇、蓍、蔖
```

#### 其他高频起名生僻字

```text
翀、赟、彧、祎、旖、旎、宓、邕、蹇、仝、烎、堃、喆、甯、绾、廸、敔、竑、宬、俣、燊、甦、龑、陞、爻、耒、翊、竔、屾、奡、郬、甪、氼、玈、黁
```

处理方式：

- 绝对禁用
- 不用"有文化""很独特""寓意好"作为保留理由
- 用户强制指定时，提示认读、书写和系统录入风险

### 4. 负面寓意不祥字

定位：

字义本身消极、阴暗、不祥，违背真实姓名寄托正向期许的基本原则。

字表：

```text
愁、殇、丧、孤、怨、厉、绝、哀、苦、病、灾、祸、亡、死、悲、恨、仇、贫、贱、穷、困、败、输、凶、毒、邪、孽、秽、垢、辱、耻、殁、殒、悼、坟、墓、棺、弃、失、落、散、离、分
```

处理方式：

- 绝对禁用
- 不作为真实姓名正向候选
- 不用反讽、悲剧感、命运感作为保留理由

### 5. 过度张扬功利字

定位：

字义过于直白外露，功利感强，不符合中式"谦受益、满招损"的含蓄审美，也与金庸式"厚重克制"相悖。

字表：

```text
霸、胜、冠、豪、强、富、贵、财、权、势、名、利、爵、禄、侯、皇、帝、圣
```

处理方式：

- 绝对禁用
- 不进入精选或备选
- 不用"大气""有出息"作为保留理由

### 6. 强江湖符号字

定位：

武侠外号、江湖习气感强，现实正式人名中过于突兀，更像绰号而非本名。

字表：

```text
侠、刀、啸、枭、狂、傲、刃、镖、舵、寇、匪、丐
```

处理方式：

- 绝对禁用
- 真实姓名场景不使用
- 角色名、网名场景需另行处理，不使用本 Skill 默认标准

### 7. 语义变迁 / 高歧义字

定位：

字义原本可能正向，但因网络文化、谐音演变或固定语义联想，容易引发低俗、不当或尴尬联想。

字表：

```text
翔、姬、芭、操
```

处理方式：

- 绝对禁用
- 避免网络语义、低俗联想和外号风险
- 具体谐音风险另见 `05B_谐音风险库.md`

## 四、慎用提示字

慎用字不是原则性错误，但不默认生成，不进入精选名。

### 1. 时代烙印过重字

定位：

不同年代的起名爆款字，带有明显时代标签，容易产生土气感、同质化感，与追求质感的起名目标不符。

字表：

```text
芬、桂、英、淑、珍、秀、花、翠、艳、丽、娟、芳、萍、建、国、军、兵、伟、福
```

说明：

- `财` 已归入"过度张扬功利字"，按绝对禁用处理
- 以上字不默认生成
- 除非用户明确偏好传统年代感，否则不使用

### 2. 甜美流行感字

定位：

字义本身正向，但现实姓名中容易显得甜、轻、网感强或模板化。不绝对禁用，但不作为优先候选，需要搭配出辨识度才可使用。

字表：

```text
泽、怡、悦、曼、薇、娴、瑶、琼、盼、笑
```

处理方式：

- 慎用
- 不进精选名
- 若用户偏好清雅温婉，也优先选择更稳的替代字
- 若使用，必须搭配有辨识度的字，避免落入常见甜美模板化组合

### 3. 气势过强字

定位：

气势感、主角感、宏大感强，容易脱离真实姓名的日常使用场景。

建议后续复核字：

```text
巍、峨、峥、嵘、瀚、阔、腾、驰、卓、越、铮、挺
```

处理方式：

- 慎用或高风险
- 不进精选名
- 需要结合姓氏和整体气质判断

### 4. 传统男名直白字

定位：

字义正向，但表达较直接，容易显得年代化、口号化或普通。

建议后续复核字：

```text
智、勇、刚、健、博、宏、勋、业
```

处理方式：

- 慎用
- 不作为优先候选
- 若使用，需要搭配更清朗、克制的字

## 五、原著强绑定字组

以下字组不直接生成真实姓名。

它们可作为金庸式命名方法对标，但不能作为最终候选名输出。

```text
无忌、婉清、芷若、语嫣、灵素、清扬、盈盈、秋水、青桐、星竹、念慈、正风、翠山、承志、百川、中则、修文、敦儒、问天、我行、莫愁、平之、坦之、退思、不群
```

处理方式：

- 不直接输出
- 不作为真实姓名候选
- 只可用于解释命名方法

注意：

单字不一定禁用。

例如：

- `清`、`文`、`修`、`慈` 可用
- `婉清`、`修文`、`念慈` 这类强绑定组合不直接生成

## 七、处理规则

1. 命中绝对禁用字：直接淘汰，不进入用户可见结果。
2. 命中高风险字：不进入精选名，只能在用户明确指定时讨论。
3. 命中慎用字：不作为优先候选，原则上不进入精选名。
4. 命中观察字：结合 `03`、`07` 和具体组合判断。
5. 命中原著强绑定字组：不得直接输出为候选名。
6. 如果用户指定禁用字，应先提示风险，再给出替代方案。
7. 禁用字不因"出处好""寓意好""看起来高级"而放行。

## 九、后续维护原则

新增禁用或风险字时，应说明：

- 风险类型
- 禁用或慎用原因
- 是否影响 `04` 现有字库
- 是否需要同步更新扩展草案

维护优先级：

1. 先处理用户明确排斥的网红字和仙侠字
2. 再补充高频生僻起名字
3. 再根据实际测试补充时代感、流行感、角色感风险字

本文件的目标不是否定所有有个性的字，而是确保真实姓名优先稳定、自然、长期可用。
