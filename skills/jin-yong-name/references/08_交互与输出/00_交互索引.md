# 08_交互与输出索引

本目录承接原 `08_输出格式与交互流程.md` 的交互、流程、硬闸门和输出模板。默认只读本索引和当前任务需要的文件。

## 先读哪个

| 场景 | 读取文件 |
|---|---|
| 用户信息不完整、需要追问 | `01_五项信息收集.md`（六项硬信息） |
| 六项齐全、准备生成 | `02_默认生成流程.md` |
| 名字长度偏好、单字名/双字名判断 | `03_名字长度规则.md` |
| 输出 3 个精选名 | `04_精选名输出模板.md` |
| 输出 2 个备选名 | `05_备选名输出模板.md` |
| 用户要求重来、调整、不满意 | `06_整体建议与二次优化.md` |
| 八字五行、禁用字、强江湖感、父母姓融合 | `07_特殊情况处理.md` |
| 输出前检查禁止项或需要完整模板 | `08_禁止输出与标准模板.md` |
| 需要确认本目录和其他 reference 的关系 | `09_文档关系.md` |

## 默认读取建议

- 起名前追问：只读 `01_五项信息收集.md`。
- 常规生成：读 `01_五项信息收集.md`、`02_默认生成流程.md`、`04_精选名输出模板.md`、`05_备选名输出模板.md`。
- 输出前硬闸门：读 `02_默认生成流程.md` 中的“输出前硬闸门”，必要时再读 `08_禁止输出与标准模板.md`。
- 特殊要求只读对应文件，不要全量读取整个目录。
