# 02_金庸方法索引

本目录承接原 `02_金庸起名方法论与案例.md` 的方法论和案例内容。生成时先读本索引，再按需读取具体方法或案例文件，不要默认全量读取所有文件。

## 先读哪个

| 场景 | 读取文件 |
|---|---|
| 需要理解金庸式起名总原则 | `01_底层逻辑.md` |
| 走经典、书卷、家学、端正路线 | `02_典籍化用法.md` |
| 女孩名、清雅、柔中有骨、诗文气质 | `03_诗词化用法.md` |
| 山水、草木、时序、风物意象 | `04_意境风物法.md` |
| 清正、温厚、明达、承担、家风 | `05_品格寄托法.md` |
| 姓氏本身有画面或需要破大姓平庸 | `06_姓氏联动法.md` |
| 姓氏短促、厚重、谐音风险高，或候选读感普通 | `07_声韵气口法.md` |
| 双字名组合、主次、声调、结构质检 | `08_双字名组合规则.md` |
| 需要参考40组人物的命名机制与展示权限 | `09_现实可用案例.md` |
| 用户要求更江湖、更金庸，或候选有角色感风险 | `10_慎用与禁用案例.md` |
| 需要判断方法能否现实转化，或输出金庸式对标表达 | `11_方法转化与表达规范.md` |

## 默认读取建议

- 普通生成：`01_底层逻辑.md` + 本轮命中的 2-3 个方法文件。
- 输出前解释：只读取与精选名对应的方法文件和 `11_方法转化与表达规范.md`。
- 不要把 `09_现实可用案例.md` 当成可复制名字库；它只提供方法对标和维度分类。
- `10_慎用与禁用案例.md` 用于降强度和避坑，不用于生成。
