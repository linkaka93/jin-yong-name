## 三补、双字名组合规则

双字名不是两个独立字并列，而是两个字组合成一个整体。本节规定组合时的判断规则。

单字筛选见 `04_常用字白名单.md`，禁用排除见 `05A`，姓氏适配见 `07`。本节只管"两个字怎么搭"。

### 1. 主次关系

- 双字名必须有主次，不能两个字都抢意义
- 主字：承载核心寓意或意象，决定名字气质方向
- 辅字：配合主字，补读音节奏，或弱化/强化气质
- 反例：两个字各自表达不同方向（如"志+远+承"三个方向都想表达，过满）

### 2. 声调搭配

- 姓+名整体看声调，有起伏优于全平或全仄
- 避免三字连续同声调（如全一声、全四声）
- 连续两个三声需慎用（普通话中会发生变调，容易拗口）
- 声母或韵母连续相近需注意（如"林立""石时"容易绕嘴）

### 3. 字义关系

两字之间的意义关系，以下三种为优：

- **相生**：辅字延伸主字意象。如"清"为主+"远"为辅，清而能远
- **互补**：辅字调和主字气质。如"毅"为主+"安"为辅，刚中有定
- **同源**：两字同出一处或同体系。如"灵素"同出《黄帝内经》

禁止：

- 两字意象重复（如"江+河""竹+松""白+雪"——画面太满，像笔名）
- 两字意义冲突（如"静+狂""柔+刚"——气质打架）
- 两字各自独立、无关联（名不成整体）

### 4. 结构偏好

常见且稳妥的双字名结构：

- 品格 + 品格：适合端正方向（如"正风""中慧"）
- 意象 + 品格：适合风物有骨（如"青桐""星竹"）
- 典籍取字：两字同出一处或同思想体系（如"灵素""修文"）
- 品格 + 意象：适合品格带画面（如"怀远""知川"）

慎用结构：

- 意象 + 意象：容易画面堆叠，变成网名或笔名感
- 功利 + 功利：容易过满，变成口号

### 5. 组合质检

双字名生成后，必须检查：

- 两个字是否有主次，不是并列
- 两个字意义是否相生或互补，不是重复或冲突
- 姓+名整体声调是否有起伏
- 换一个字是否明显更好——如果是，说明当前组合不够稳
