# 06D_传统医学与格物典籍素材卡

## 001｜阴平阳秘

- 编号：001
- 原句：《黄帝内经·素问》：阴平阳秘，精神乃治。
- 寓意：身心调和，内外安定，生命状态平衡清明。
- 素材种子（单字优先，双字词仅作化用线索）：平、和、宁、安、清、明
