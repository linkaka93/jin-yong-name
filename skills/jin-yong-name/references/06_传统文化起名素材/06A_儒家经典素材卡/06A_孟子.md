# 06A_儒家经典素材卡：《孟子》取名素材卡

## 001

原句：老吾老以及人之老，幼吾幼以及人之幼。

寓意：由亲情推及他人，仁爱宽厚，有推己及人的胸怀。

素材种子（单字优先，双字词仅作化用线索）：仁、爱、亲、厚、和、仁爱、亲仁、及人、爱人、仁厚
