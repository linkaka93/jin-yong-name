# 05B_谐音风险库

## 一、文档定位

本文件是「金庸帮你起名字」Skill 的姓名读音与连读风险检查库。

它用于检查姓名全名在普通话连读时是否存在谐音、误听、外号、网络梗、低俗联想、贬义联想或方言提示风险。

本文件只处理声音层面的风险。

本文件不处理：

- 字义禁用：交给 `05A_禁用字与风险字库.md`
- 常用字选择：交给 `04_常用字白名单.md`
- 姓氏适配：交给 `07_姓氏适配规则.md`
- 文化出处：交给 `06_传统文化起名素材/`
- 最终质检：交给 `03_好名字评判标准.md`

核心原则：

> 名字单独好听不够，必须连姓一起读。只要全名连读后形成明显不良谐音，就应淘汰。

---

## 二、使用边界

检查对象：

- 姓 + 单字名
- 姓 + 双字名
- 全名普通话连读
- 常见误听
- 常见外号风险
- 必要时提示方言风险

默认判断口径：

- 以普通话为主
- 只拦截高频、明显、现实中容易被调侃的风险
- 不追求覆盖所有小众谐音
- 不把单个音节机械判死，必须结合全名连读判断

方言处理：

- 用户未说明地域时，不默认展开方言判断
- 用户提供地区或家庭语言环境时，可提示当地常见风险
- 方言风险通常列为“慎用”或“提示”，除非风险非常明显

---

## 三、风险等级

### 1. 绝对禁用

触碰低俗不雅、强侮辱、灾病死亡、严重人格伤害的谐音。

处理方式：

- 直接淘汰
- 不进入精选名
- 不进入备选名
- 不向用户展示为候选

### 2. 高风险

存在明显贬义、强外号风险、网络固定负面联想。

处理方式：

- 不进入精选名
- 原则上淘汰
- 用户强制要求时，必须提示风险并给替代方案

### 3. 慎用

存在尴尬歧义、轻微误听、方言风险或容易被起外号。

处理方式：

- 原则上不进入精选名
- 可作为内部备选观察
- 若输出，必须在现实评估中说明风险

### 4. 提示

仅有弱联想，本名寓意正当，且普通话连读风险不明显。

处理方式：

- 可保留
- 在现实评估中轻提示
- 不作为直接淘汰依据

---

## 四、谐音检查流程

对每个候选姓名按以下顺序检查：

1. 全名普通话连续读 3 遍。
2. 检查是否接近低俗不雅词。
3. 检查是否接近贬义侮辱词。
4. 检查是否接近疾病、灾祸、贫穷、失败类词。
5. 检查是否容易被误听成外号或网络梗。
6. 检查姓氏是否属于高风险姓氏。
7. 检查是否存在方言提示风险。
8. 根据风险等级决定通过、慎用或淘汰。

---

## 五、高风险谐音类型

### 1. 低俗不雅类

定位：

触碰公序良俗底线，容易引发恶意调侃，是起名的绝对红线。

核心风险词汇：

| 拼音 | 风险词 / 联想 |
|---|---|
| shi | 屎、屎尿、屎壳郎 |
| niao | 尿、尿床 |
| pi | 屁、放屁、屁股 |
| gang | 肛门 |
| diao | 屌、吊丝 |
| bi | 逼、傻逼 |
| yang wei | 阳痿 |
| liu chan | 流产 |
| fu yan jie | 妇炎洁 |
| gao wan | 睾丸 |
| ji ba | 鸡巴 |
| fan tong | 饭桶 |
| du zi teng | 肚子疼 |
| shi zhen xiang | 屎真香 |

典型踩坑姓名：

```text
史珍香、史尚飞、杜子腾、杜琦燕、范统、范建、杨伟、杨委、刘产、付岩杰、朱逸群、秦寿生、沈京兵、杜蕾斯、史一坨、吴耀、魏生津
```

处理方式：

- 绝对禁用
- 直接淘汰
- 不作为候选展示

### 2. 贬义侮辱类

定位：

谐音直接指向智力、人品、命运或行为的负面评价，容易造成长期人格伤害。

核心风险词汇：

| 类型 | 风险词 / 联想 |
|---|---|
| 智力类 | 呆、傻、笨、蠢、白痴、弱智 |
| 人品类 | 贱、坏、恶、无德、无用、无良、假正经、不要脸 |
| 命运类 | 穷、惨、丧、病、灾、祸、无能、不幸 |
| 行为类 | 滚、输、败、骗、偷 |

典型踩坑姓名：

```text
吴德、吴用、吴勇、贾正经、贾仁义、梅良心、莫有仁、胡丽晶、熊初墨、卜是人、魏生金、李无能、张不幸、王败类
```

处理方式：

- 绝对禁用或高风险
- 明显侮辱性谐音直接淘汰
- 弱联想也不进入精选名

### 3. 尴尬歧义类

定位：

没有强烈侮辱性，但容易被起外号、产生不当联想，日常社交中容易尴尬。

核心风险词汇：

| 类型 | 风险词 / 联想 |
|---|---|
| 动物类 | 猪、狗、驴、鸡、鸭、猴、鼠、虫 |
| 状态类 | 疯、癫、骚、浪、短、大头 |
| 事物类 | 桶、瓶、罐、医院、炎症、肺炎 |
| 其他 | 无情、东西、你猜 |

典型踩坑姓名：

```text
朱一群、侯丹、姬从良、史定、吴晴、李猜、董西、费彦、严育、王小二、张大炮、刘鼻涕
```

处理方式：

- 慎用或高风险
- 外号风险强时淘汰
- 不进入精选名

### 4. 网络衍生贬义类

定位：

字或音本身未必负面，但因网络文化传播产生固定贬义联想，属于“无妄之灾”类风险。

核心风险词汇：

| 拼音 / 音节 | 风险词 / 联想 |
|---|---|
| xiang | 翔，网络语境中可代指粪便 |
| ji | 姬、基，部分网络语境有贬义或调侃联想 |
| cai | 菜，代指能力差、菜鸟 |
| la ji | 垃圾 |
| ge | 鸽，代指爽约、放鸽子 |
| diao | 掉，掉链子、差劲联想 |

典型踩坑姓名：

```text
李翔、王翔、张姬、蔡袅、腊鸡、葛优、刁婵
```

处理方式：

- 高风险或慎用
- 网络固定负面联想强时淘汰
- 普通话无明显风险但网络语义较强时，不进入精选名

### 5. 疾病灾祸类

定位：

全名连读后接近疾病、死亡、灾害、痛苦等负面词，违背真实姓名的正向期许。

核心风险词汇：

```text
病、亡、死、伤、灾、祸、残、惨、苦、哭、流产、肺炎、炎症、肚子疼
```

高风险音节：

```text
bing、wang、si、shang、zai、huo、can、ku、ku、liu chan、fei yan、yan zheng、du zi teng
```

处理方式：

- 绝对禁用或高风险
- 明显疾病灾祸谐音直接淘汰

### 6. 贫穷失败类

定位：

全名连读后接近贫穷、失败、无能、倒霉等消极词，容易形成长期负面心理暗示和外号风险。

核心风险词汇：

```text
穷、贫、败、输、废、没用、无能、不幸、倒霉、失败、饭桶
```

高风险音节：

```text
qiong、pin、bai、shu、fei、mei yong、wu neng、bu xing、dao mei、shi bai、fan tong
```

处理方式：

- 高风险
- 明显负面谐音直接淘汰
- 弱联想不进入精选名

## 八、常见连读风险模式

### 1. 姓尾音 + 名首音连成负面词

名字单独看没有问题，但姓和名连在一起后形成负面词。

处理方式：

- 全名必须连读检查
- 不能只看名字两个字

### 2. 全名听起来像低俗词或骂人话

全名整体音节接近低俗词、脏话或侮辱性表达。

处理方式：

- 绝对禁用
- 直接淘汰

### 3. 名字单独好，连姓后变味

名字本身寓意正向，但连姓后产生“无、没、假、屎、猪、废”等误听。

处理方式：

- 高风险姓氏需要更严格检查
- 不能用名字本身寓意正当来抵消明显谐音

### 4. 同音、近音、变调导致误听

普通话中声调不同但音节接近，日常喊叫时可能被误听。

处理方式：

- 风险强时淘汰
- 风险弱时慎用

### 5. 网络语义变迁导致外号风险

某些字或音在网络语境中已有固定调侃含义。

处理方式：

- 网络固定负面联想强时淘汰
- 弱联想则提示

### 6. 方言环境下产生额外风险

普通话无风险，但家庭、学校或当地常用方言中存在明显不良谐音。

处理方式：

- 用户未提供地区时，只作一般提示
- 用户提供地区时，应额外复核

## 十、处理规则

1. 命中绝对禁用谐音：直接淘汰。
2. 命中高风险谐音：不进入精选名，原则上淘汰。
3. 命中慎用谐音：不进入精选名，可作为内部备选观察。
4. 命中提示级弱联想：可保留，但在现实评估中简要说明。
5. 用户指定的名字命中谐音风险时，先说明风险，再给替代方案。
6. 名字字义再好，也不能抵消明显不良谐音。
7. 无法确认的方言风险，不硬判，可提示用户结合当地读音复核。

## 十一、输出表达规范

最终输出不展示完整谐音排查过程。

可用表达：

- 全名普通话连读顺，无明显不良谐音风险。
- 普通话读音可用，但如果家庭常用方言较多，建议再用当地读音确认。
- 这个名字连姓后存在轻微误听可能，因此不放入精选，只作为备选观察。

避免表达：

- 绝对没有任何谐音风险。
- 这个名字所有方言都没问题。
- 这个名字谐音大吉。
- 为了展示风险而向用户列出低俗反例。

## 十三、后续维护原则

新增谐音风险时，应记录：

- 风险音节
- 风险词
- 风险等级
- 典型连读模式
- 是否只在特定方言中成立

维护优先级：

1. 先补充高频低俗不雅风险。
2. 再补充强侮辱、疾病灾祸、贫穷失败类风险。
3. 再补充网络语义变迁风险。
4. 最后根据真实起名测试补充姓氏连读案例。

本文件目标不是制造焦虑，而是提前排除现实生活中最容易被调侃、误听和伤害的读音风险。
