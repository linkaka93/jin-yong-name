# 07_姓氏适配规则

## 一、文档定位

本文件用于判断姓氏与名字是否整体适配。

它解决的问题是：

- 这个姓本身是什么气质
- 这个姓适合什么命名策略
- 名字是否能和姓连成一个整体
- 是否存在读音、字义、气质上的冲突

本文件不处理：

- 单字是否可用：见 `04_常用字白名单.md`
- 禁用字和风险字：见 `05A_禁用字与风险字库.md`
- 谐音风险：见 `05B_谐音风险库.md`
- 文化出处：见 `06_传统文化起名素材/`
- 最终输出格式：见 `08_交互与输出/00_交互索引.md`

核心原则：

> 姓氏不是名字前面的独立前缀，而是全名气质的一部分。

---

## 二、姓氏适配的判断顺序

生成或质检一个名字时，按以下顺序判断：

1. 看姓氏类型
   - 大姓、小姓、复姓、意象姓、厚重姓、轻柔姓、短促姓、强谐音风险姓。

2. 看读音节奏
   - 姓和名连读是否顺口，声调是否过挤，是否容易误听。

3. 看字义关系
   - 姓氏字义是否能与名字形成联动，是否有重复、冲突或尴尬联想。

4. 看气质平衡
   - 姓氏已经厚重，名字不宜再厚；姓氏偏轻，名字可以补骨力。

5. 看现实使用
   - 全名是否像真实姓名，是否过于小说化、网文化、口号化。

6. 看可替代性
   - 换一个姓后名字是否质感明显下降。若下降，说明姓氏适配较强。

---

## 三、姓氏类型与命名策略

### 0. 姓氏多类型归属的优先级

某些姓氏可能同时属于多个类型（如"白"既是意象姓又是短促姓，"林"既是大姓又是意象姓，"史"既是短促姓又是高谐音风险姓）。按以下优先级判断主类型：

1. 高谐音风险姓 → 优先按高谐音风险处理（安全第一）
2. 复姓 → 按复姓处理（风格强度必须降低）
3. 其余按姓氏最突出特征判断：
   - 意象姓（字义画面感强）优先按意象姓
   - 厚重 / 轻柔 / 短促按声韵气质判断
   - 大姓单独处理（破俗需求最独特）

### 1. 大姓

常见姓氏：

```text
王、李、张、刘、陈、杨、赵、黄、周、吴、徐、孙、胡、朱、高、林、何、郭、马、罗
```

特点：

- 使用人数多，普通组合容易重名
- 名字太常规会显得平
- 名字太用力又容易变网红、仙侠或口号化

命名策略：

- 用结构破俗，不用生僻字破俗
- 选择常用字中有厚度的组合
- 适合书卷、端正、开阔、温厚方向
- 避免高频网红字和模板化组合

适合方法：

- 大姓破局
- 品格寄托
- 典籍化用
- 声韵气口法

示例对标：

- 张无忌式：大姓用平实字做出厚度
- 王语嫣式：大姓通过名字柔化气质
- 郭靖式：大姓搭配单字名时要有足够力量

注意：

- 大姓不宜配过普通的热门字
- 大姓也不宜为了独特使用生僻字
- 大姓的名字解释必须有内核，不能只靠“好听”

### 2. 字义强的意象姓

常见姓氏：

```text
林、江、叶、白、夏、温、柳、石、方、云、谷、田、池、梅、苗
```

特点：

- 姓氏本身有画面或字义
- 很适合做姓氏联动
- 起得好会有整体感，起得不好容易意象重复

命名策略：

- 顺着姓氏画面展开
- 名字补充画面，不重复画面
- 避免“姓和名都在说同一件事”
- 不要堆太多风物字

适合方法：

- 姓氏联动法
- 意境风物法
- 诗词化用法

示例对标：

- 木婉清式：姓氏与名字气质自然贴合
- 风清扬式：姓氏带动整体意境
- 阮星竹式：姓氏与风物画面协调

注意：

- 林姓不必再堆太多草木字
- 江姓不必再堆水意象
- 白姓慎用过冷、过素的字
- 温姓慎用过柔的字

### 3. 厚重姓

常见姓氏：

```text
郭、陆、沈、韩、曹、董、邓、程、贺、袁、梁、薛、蒋、曾
```

特点：

- 声音较厚、字形较实
- 容易显稳重，也容易显沉
- 名字不宜再堆过重字

命名策略：

- 用清朗、明净、书卷字调和
- 可用山水、开阔字舒展气口
- 避免厚上加厚

适合方向：

- 清朗书卷
- 简净温润
- 疏朗有风骨
- 开阔大气

慎用方向：

- 过重的德、厚、承、志叠加
- 过硬的刚、毅、锋、峥类字
- 过庄重的传统男名组合

### 4. 轻柔姓

常见姓氏：

```text
苏、许、柳、温、叶、乔、顾、夏、白、云、莫
```

特点：

- 声音轻、画面柔
- 适合清雅、温润、书卷方向
- 也容易过甜、过飘、过软

命名策略：

- 可适当加入骨力字
- 保留清雅，但避免甜腻
- 女名不宜过度柔化
- 中性名可走简净、明达方向

适合方向：

- 清雅温婉
- 简净温润
- 柔中有骨
- 清朗书卷

可用调和：

- 衡、正、仪、立、清、安、知、远

注意：

- 不要连用太多柔、婉、宁、若类轻柔字
- 不要变成古风网名或艺名

### 5. 短促姓

常见姓氏：

```text
丁、马、方、石、白、孔、史、毕、卜、陆
```

特点：

- 姓氏短、硬、气口明显
- 名字需要顺口舒展
- 单字名要求更高

命名策略：

- 双字名更稳
- 名字宜有开合变化
- 避免再配短促、硬直字
- 必须重点检查谐音

适合方向：

- 清朗书卷
- 简净温润
- 开阔大气

注意：

- 短促姓 + 单字名容易太硬
- 石、史、施等姓需重点查 `05B`

### 6. 复姓

常见姓氏：

```text
欧阳、司马、上官、诸葛、东方、慕容、令狐、夏侯、皇甫
```

特点：

- 姓氏已经有很强存在感
- 名字不宜复杂
- 过度风格化会像小说人物

命名策略：

- 名字宜简
- 优先单字名或简洁双字名
- 避免过古风、过仙侠、过角色化
- 不堆典故，不堆意象

适合方向：

- 简净温润
- 端方有度
- 清朗书卷

注意：

- 复姓天然容易小说化，必须降低风格强度
- 复姓不要配玄、霄、卿、辞、弦等风格字

### 7. 高谐音风险姓

常见姓氏：

```text
史、施、石、贾、杜、吴、梅、莫、朱、牛、马、苟、赖、刁、焦、毕、郝、傅、侯、夏、殷、尹、范、杨、刘、付、魏、秦、沈、胡、熊、卜、费、严、董、蔡、葛
```

特点：

- 姓氏本身容易和负面词、否定词、外号产生连读风险
- 起名时要先查谐音，再看寓意

命名策略：

- 必须优先查 `05B_谐音风险库.md`
- 避免容易连读成负面词的字
- 名字再好听，也不能抵消明显谐音风险

注意：

- 高风险姓不是不能起好名
- 只是需要更严格的连读检查

## 四、声调与气口适配

### 1. 基本原则

全名读起来应当自然、有起伏、不堵、不飘。

判断方式：

- 连续读 3 遍
- 写出声调序列，例如 `2-3-4`
- 三字全名若为 `1-1-1`、`2-2-2`、`3-3-3`、`4-4-4`，直接淘汰
- 三字全名若全为平声，即只由一声、二声组成，例如 `2-1-2`、`2-2-1`、`1-2-1`，不得只因“不是同调”而放行，精选名应优先改成有平仄起伏的结构
- 看是否卡口
- 看是否容易误听
- 看日常喊叫是否自然

### 2. 常见问题

```text
全名三字同调：没有起伏，例如 1-1-1、2-2-2、3-3-3、4-4-4
全名三字全平：不是同调但仍偏平，例如 2-1-2、2-2-1、1-2-1
全名声调太平：读起来没起伏
全名声调太挤：读起来压迫、卡顿
连续三声：容易变调或拗口
声母连续相近：容易绕嘴
韵母连续相近：容易黏连
```

### 3. 处理规则

- 声调问题轻微：可慎用
- 三字同调：直接淘汰，不列为慎用
- 三字全平：不得进入精选名；除非整体极稳，也只能列为慎用备选
- 明显拗口：淘汰
- 连读产生谐音：按 `05B` 处理
- 读音不如寓意重要：错误；真实姓名必须先能叫

## 五、字义与意象适配

### 1. 姓名意义要相生，不要相撞

通过：

- 姓氏和名字意义能形成整体
- 名字补充姓氏气质
- 全名有画面或品格方向

淘汰：

- 姓和名意义冲突
- 姓和名意义重复
- 全名解释牵强
- 名字脱离姓氏单独成立，但全名不成立

### 2. 避免意象堆叠

不建议：

```text
林 + 竹 + 松
江 + 河 + 海
白 + 雪 + 霜
云 + 霄 + 岚
```

原因：

- 画面太满
- 容易像笔名或网名
- 缺少真实姓名的克制感

### 3. 避免气质打架

常见冲突：

```text
厚重姓 + 过重名字
轻柔姓 + 过软名字
复姓 + 古风名字
短促姓 + 硬直单字名
高谐音风险姓 + 易误听名字
```

## 六、单字名与双字名适配

### 1. 单字名适用条件

单字名可以用，但要求更高。

适用条件：

- 姓氏读音稳定
- 单字含义足够厚
- 全名读音干净
- 无明显谐音风险
- 不显单薄

适合：

- 部分大姓
- 厚重姓
- 气口稳定的姓

慎用：

- 短促姓
- 高谐音风险姓
- 复姓
- 姓氏本身太轻的姓

### 2. 双字名适用条件

双字名是默认优先方案。

适用条件：

- 需要补足姓氏气质
- 需要表达更完整的期许
- 需要降低重名感
- 需要调和读音节奏

注意：

- 双字名不能堆砌
- 两个字必须有主次
- 不要两个字都抢意义

## 七、姓氏适配结论类型

### 1. 高适配

判断标准：

- 姓和名读音顺
- 字义能联动
- 气质协调
- 全名有整体感
- 换姓后质感明显下降

可进入精选。

### 2. 中适配

判断标准：

- 全名可用
- 读音无明显问题
- 字义不冲突
- 但姓氏联动不强

可进入备选。

### 3. 低适配

判断标准：

- 名字本身不错，但和姓氏关系弱
- 读音或气质不够协调
- 全名没有整体感

原则上不输出。

### 4. 不适配

判断标准：

- 读音明显不顺
- 谐音风险明显
- 字义冲突
- 气质打架
- 现实使用突兀

直接淘汰。

## 八、生成时的使用规则

1. 先判断姓氏类型，再选择命名方法。
2. 大姓先破俗，小姓先放大气质。
3. 意象姓先看能否做姓氏联动。
4. 厚重姓用清朗字调和。
5. 轻柔姓适当补骨力。
6. 复姓必须降低风格强度。
7. 高谐音风险姓先查 `05B`。
8. 单字名不强推，必须更严格筛选。
9. 姓氏适配不通过的名字，不进入最终输出。
10. 性别倾向与用字配合：
    - 用户要男孩名：优先用"男/中性"字，可少量用"中性"字做调和
    - 用户要女孩名：优先用"女/中性"字，可用少量"男/中性"字补骨力（柔中有骨方向）
    - 用户要中性名：优先用"中性"字，避免性别倾向过强的字
    - 不机械执行：某个"女"字搭配姓氏后气质偏中性，可灵活使用

## 九、输出时的表达方式

最终输出中不展示完整姓氏分析，只在“姓氏适配”项中简洁说明。

可用表达：

- “陈”姓稳重，搭配这个名字后有书卷气，不显老气。
- “林”姓自带风物感，名字没有再堆草木意象，整体清雅但不重复。
- “王”是大姓，这个名字用常见字做出辨识度，不靠生僻字取巧。
- “温”姓本身柔和，名字补了一点骨力，避免过甜。

避免表达：

- 这个姓天生不好起。
- 这个姓不适合好名字。
- 这个名字换什么姓都一样好。
- 只解释名字，不解释全名。
