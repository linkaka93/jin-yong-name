# 04_常用字白名单

## 一、文档定位

本文件是「金庸帮你起名字」Skill 的现实可用字校验库。

它用于候选名形成之后，检查单字是否稳定、常见、正向、现实可用，减少生僻化、网红化、仙侠化和空泛化风险。

本文件不是完整汉字表，也不是唯一可用字表，更不是生成姓名的方向来源。

核心作用：

- 校验候选名用字是否常见、稳妥、现实可用
- 在候选已有金庸式方法和姓氏整体感之后，提供补字参考
- 提醒哪些常用字容易造成模板化、安全但无趣的组合

本文件不再作为第一轮灵感生成库。生成名字时，应先由 `07`、`09`、`02` 和必要的 `06` 决定姓名结构、金庸式方法和文化方向，再用本文件检查单字可用性。

反误用规则：

- `04` 只能辅助判断“这个字能不能用”，不能决定“这个名字该往哪里走”。
- 不得从 `04` 的标签、补字参考或索引出发批量拼名。
- 若候选名只是白名单字的安全组合，但没有 `02` 方法、姓氏参与或 `06` 素材支撑，应直接淘汰。
- 高频安全字不得连续复用；`远`、`川`、`知`、`行`、`怀`、`明`、`安`、`衡`、`承`、`朗` 等字进入精选名时，必须证明它不是模板化替换。

---

## 二、使用边界

本文件只判断单字的基础可用性。

不在本文件处理：

- 姓氏适配：统一交给 `07_姓氏适配规则.md`
- 禁用字、风险字、网红字、仙侠字：统一交给 `05A_禁用字与风险字库.md`
- 谐音风险：统一交给 `05B_谐音风险库.md`
- 最终质检：统一交给 `03_好名字评判标准.md`
- 具体典籍出处和素材卡：统一交给 `06_传统文化起名素材/`

使用顺序：

1. 先用 `07` 判断姓氏，用 `09` 路由，用 `02` 选择金庸式方法，必要时用 `06` 提供文化素材。
2. 候选名已经形成后，再用本文件检查单字是否常见、易认、易写、字义稳定。
3. 若某个候选只因白名单搭配而成立，没有姓氏参与和金庸式方法，应淘汰。
4. 若候选不足，可参考扩展常用字草案，但必须等待 `05A` 复核。
5. 候选名字必须继续经过 `05A`、`05B`、`03` 检查。

## 三、入选标准

精选可用字必须同时满足：

- 常见易认
- 日常易写
- 字义正向或中性偏正向
- 适合真实姓名
- 能承载清晰气质
- 不依赖冷门典故
- 不网红
- 不仙侠
- 不猎奇
- 不明显负面

优先收入：

- 常用但有余味的字
- 字义稳、解释空间清楚的字
- 可用于男孩、女孩或中性姓名的字
- 能服务金庸式“浅字深用”的字

---

## 四、排除标准

本文件不收入以下类型：

- 生僻字
- 异体字
- 难写难认字
- 近年高频网红伪风雅字
- 仙侠、网文、游戏感过强的字
- 负面、阴冷、病弱、狠厉、怪异的字
- 过满的成功学字
- 与金庸原著人物强绑定、容易引发仿写感的字组

已明确避开的高风险字包括但不限于：

- 梓、涵、轩、墨、辰、宸、诺、汐、沐、卿、兮
- 玄、尘、烬、渊、辞、弦、玦、璟、珩

这些字不进入精选可用字表。

## 五、字段说明

精选可用字表字段固定如下：

| 字段 | 说明 |
|---|---|
| 字 | 单个汉字 |
| 拼音 | 普通话拼音，不标声调 |
| 基础含义 | 字的稳定基础义，不做过度阐释 |
| 气质标签 | 用于校验和补字参考，可有多个标签 |
| 性别倾向 | 男 / 女 / 中性 / 男/中性 / 女/中性 |
| 可用倾向 | 用于判断字感是否过偏，不作为用户默认选项 |
| 补字参考 | 仅作候选替换参考，不得直接按此批量生成名字 |
| 使用注意 | 生成时需要避免的问题 |

气质标签不是唯一分类。

同一个字可以有多个标签，例如：

- 清：清雅、明净、端正、简净
- 衡：端正、沉稳、有骨、从容
- 远：开阔、书卷、沉稳、疏朗

## 九、使用规则

1. 精选可用字不是默认生成入口，只能作为候选校验和补字参考。
2. 一个名字不应只因为用了白名单字就直接通过。
3. 不得直接按“补字参考”批量生成候选名。
4. 所有候选名必须继续经过 `05A_禁用字与风险字库.md`、`05B_谐音风险库.md` 和 `03_好名字评判标准.md`。
5. 若候选字命中 `05A_禁用字与风险字库.md`，以 `05A` 为准。
6. 若全名存在谐音风险，以 `05B_谐音风险库.md` 为准。
7. 字与姓氏是否适配，以 `07_姓氏适配规则.md` 为准。
8. 扩展常用字草案只作为后续扩库参考，不作为默认优先用字。
9. 如果候选池因为本文件变得同质，应回到 `02` 和 `06` 重新生成，而不是继续在本文件中找近义替换字。

## 十、后续扩展原则

新增精选字时，必须满足：

- 字形常见
- 读音清楚
- 字义稳定
- 适合真实姓名
- 不在 `05A` 禁用或高风险范围内
- 能补充现有标签覆盖
- 字段完整

扩展顺序建议：

1. 先通过真实起名测试发现缺口。
2. 再从扩展常用字草案中筛选。
3. 经 `05A` 复核后，补入精选可用字表。
4. 同步更新气质标签校验索引。

第一版目标不是覆盖所有可用汉字，而是让 Skill 在生成时更稳、更克制、更接近真实姓名。
